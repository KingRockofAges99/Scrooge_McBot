## 🛠 **How to Use Scrooge_McBot**
1. Clone this repository:
   ```bash
   git clone https://github.com/KingRockofAges99/Scrooge_McBot.git
   ```

2. Install the required dependencies:
   ```bash
   pip install -r dependencies.txt
   ```

3. Configure your **phantom_wallet.json** file and **bot_configuration.json**.

4. Run the bot:
   ```bash
   python botcode.py
   ```

5. Visit the web interface at **http://localhost:5000** to monitor and execute trades.

## 📈 **How It Works**
1. **Monitors DexScreener** for newly listed meme coins.
2. **Tracks trends from Pump.fun** to identify promising tokens.
3. **Executes trades automatically** using your Phantom Wallet.
